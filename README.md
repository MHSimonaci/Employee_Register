# Employee Salary Management System

Java application for managing employee registration and salary updates using Object-Oriented Programming principles.

The system allows registering employees with unique IDs, names, and salaries, preventing duplicate IDs to ensure data integrity. It also supports salary increases based on percentage calculations and updates employee information accordingly.

## Features

* Employee registration
* Unique ID validation
* Salary increase by percentage
* Updated employee listing
* Encapsulation with private attributes
* Data handling using ArrayList

## Technologies

* Java
* Object-Oriented Programming (OOP)
* Java Streams API
* ArrayList

## Concepts Practiced

* Encapsulation
* Constructors
* Getters and setters
* Methods
* Loops and conditionals
* List manipulation
* Stream filtering

This project was built to reinforce core Java fundamentals and improve understanding of object-oriented design, data handling, and clean code structure.
